
**Behavior:**
- Auto-focus first input
- Auto-advance to next input on type
- Auto-submit when 6 digits entered
- Countdown timer visible
- Resend button disabled for 60 seconds
- Rate limit: Max 3 OTP requests per hour

**On Submit:**
- Call API: `POST /api/verify-otp`
- Store auth token
- Redirect to onboarding

## MOBILE-FIRST CONSIDERATIONS

### Touch Targets
- All buttons minimum 44x44px
- Adequate spacing (8px+) between elements
- No hover states (use active states)

### Performance
- Lazy load images below fold
- Use WebP with JPG fallback
- Minimal JavaScript
- Target <2s load time

### Navigation
- Sticky header on scroll
- Bottom navigation for key actions
- Easy thumb reach for primary CTAs
