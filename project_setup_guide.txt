## ðŸ› Common Issues & Solutions

### Issue: Image compression too slow
**Solution:** Use queue jobs for async processing
```php
dispatch(new CompressProductImage($product));
```

### Issue: OTP not received
**Solutions:**
1. Check Zenziva balance
2. Verify phone number format (08xxx)
3. Check sandbox mode disabled
4. Rate limit exceeded?

### Issue: CORS errors
**Solution:** Update `config/cors.php`
```php
'paths' => ['api/*'],
'allowed_origins' => ['http://localhost:5173', 'https://katalogku.id'],
'supports_credentials' => true,
```

### Issue: Storage symlink not working on shared hosting
**Solution:** Manual symlink via cPanel or FTP
```
public_html/storage -> ../../storage/app/public
```
